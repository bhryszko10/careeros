# CareerOS - Job Application Tracker

The command center for modern job seekers. Track applications, nail negotiations, and map your career path.

## Quick Start

### Prerequisites
- Node.js 18+ installed
- npm or pnpm

### Setup

1. Open terminal in this folder:
```bash
cd C:\Users\brett\OneDrive\Documents\Claude_Access\CareerOS_Project
```

2. Install dependencies:
```bash
npm install
```

3. Start development server:
```bash
npm run dev
```

4. Open http://localhost:5173 in your browser

### Build for Production

```bash
npm run build
```

Output will be in the `dist/` folder.

## Features

- **Visual Pipeline**: Kanban-style board with drag & drop
- **Job Tracking**: Company, position, salary, contacts, notes
- **Search & Filter**: Find jobs instantly
- **Analytics**: Response rate tracking
- **Local Storage**: Data persists in your browser

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui components
- LocalStorage for persistence
